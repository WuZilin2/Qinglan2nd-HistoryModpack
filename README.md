## 📖 简介

**轻蓝2nd** 是一个以 **城市建设** 和 **轨道交通模拟** 为核心的 Minecraft Fabric 整合包。它集成了机械动力、MTR 模组及大量装饰与辅助工具，旨在为玩家提供一个流畅、便捷且富有创造力的建筑与交通模拟环境。

本包内置了全面的性能优化模组，确保在丰富的模组内容下依然能获得稳定的游戏体验。

## 🎮 核心玩法

- **轨道交通模拟**：基于 `Minecraft Transit Railway (MTR)`，包含伦敦地铁、天津地铁、俄罗斯地铁等丰富扩展，体验逼真的列车驾驶与地铁站务管理。
- **机械动力与自动化**：利用 `Create` 及其附属模组（伪装方块、内饰、装饰）搭建精密的机械装置和动态装饰。
- **城市建设与装饰**：通过 `方块小镇 (Yuushya)`、`迷上城建 (Mishang UC)`、`MrCrayfish 的家具` 等模组，自由构建现代都市景观。
- **高效建造辅助**：集成 `Axiom`、`创世神 (WorldEdit)`、`投影 (Litematica)`、`轻松建造 (Effortless Building)` 等工具，大幅提升大型建筑效率。

## ⚙️ 安装与启动

### 1. 环境要求
- **Minecraft 版本**：1.20.1
- **模组加载器**：Fabric 0.16.14+ (已内置)
- **Java 版本**：**Java 17** (推荐 Zulu JDK 17、Eclipse Temurin 17 或 Amazon Corretto 17)

### 2. 安装步骤
1. 将整合包文件夹（包含 `.minecraft`、启动器等）解压至任意目录。
2. 启动 PCL / HMCL / 官方启动器，选择 `Qinglan2ndModPack_UnofficialVersion` 版本。
3. 分配至少 **8GB** 内存（推荐 **8-16GB**，根据物理内存调整，不要超过总内存的 70%）。
4. 启动游戏。
